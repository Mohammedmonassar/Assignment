📝 فلاتر Django Template Language (DTL)

فلاتر النصوص

الفلتر الوظيفة
lower تحويل النص إلى أحرف صغيرة
upper تحويل النص إلى أحرف كبيرة
title تحويل أول حرف من كل كلمة إلى كبير
length إرجاع عدد الأحرف
truncatechars:10 قص النص بعد عدد معين من الأحرف

🔢 فلاتر الأرقام

الفلتر الوظيفة
add:"5" إضافة رقم للقيمة
divisibleby:"3" التحقق مما إذا كانت القيمة تقبل القسمة على رقم معين

⏳ فلاتر التاريخ والوقت

الفلتر الوظيفة
date:"Y-m-d" تنسيق التاريخ
time:"H:i" تنسيق الوقت
timesince عرض الفرق الزمني منذ تاريخ معين

⚙️ فلاتر منطقية / شرطية

الفلتر الوظيفة
default:"N/A" عرض قيمة افتراضية إذا كانت None
yesno:"نعم,لا,ربما" تحويل القيمة إلى نص مخصص

📂 فلاتر القوائم والمجموعات

الفلتر الوظيفة
first عرض أول عنصر
last عرض آخر عنصر
join:", " دمج عناصر القائمة بفاصل معين
slice:":3" عرض أول ثلاث عناصر

🔧 طريقة التفعيل في مشروع Django

# فلاتر DTL تعمل افتراضيًا بدون تثبيت.

# لتفعيل Jinja2:
- أضف Jinja2 إلى `settings.py` في قسم `TEMPLATES`.
- ركّب المكتبة إن لم تكن موجودة: `pip install Jinja2`

# لتفعيل Mako:
- ثبّت الحزمة: `pip install mako`
- ثم إعداد المحرك داخل ملفات الإعدادات حسب المكتبة المستخدمة.

🆚 مقارنة بين DTL وJinja2 وMako في Django

الميزة DTL (الافتراضي) Jinja2 Mako
السرعة جيدة أسرع من DTL سريع جدًا
سهولة التعلم سهل وبسيط متوسط معقد نوعًا ما
درجة الأمان عالي جدًا جيد (فلترة يدوية) أقل أمانًا

حرية التعبير مقيد (لحماية المستخدم) حر (يشبه Python) كود حر جدًا
الملحقات والدوال محدودة كثيرة ومتقدمة متقدمة
التوثيق والمجتمع ممتاز ممتاز متوسط
التخصيص محدود قابل للتخصيص عالي قابل للتخصيص جدًا

🔧 كيفية تفعيل كل محرك قوالب

### 1. تفعيل DTL (الافتراضي)
لا يحتاج إعداد إضافي. يكون مفعّل تلقائيًا في Django:
'BACKEND': 'django.template.backends.django.DjangoTemplates'
### 2. تفعيل Jinja2
الخطوات:
- تثبيت: `pip install Jinja2`
- إضافة إعدادات في `settings.py`
- مثال:
'BACKEND': 'django.template.backends.jinja2.Jinja2'
### 3. تفعيل Mako
الخطوات:
- تثبيت: `pip install mako`
- استخدام مكتبة `django-mako-plus`
- إضافة إعدادات وربطها
